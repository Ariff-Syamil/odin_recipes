Ayam Masak Lemak Cili Api

Ayam Masak Lemak Cili Api dihidang bersama Kentang

Makanan kesukaan rakyat Malaysia

Makanan dari etnik India di Malaysia

Ingredients

  • Seekor Ayam
  • Sebungkus Serbuk Ayam Masak Lemak Cili Api Babas
  • Kentang

Steps to cook Ayam Masak Lemak Cili Api

  1. Basuh ayam
  2. Panaskan minyak dan masukkan serbuk Ayam Masak Lemak Cili Api Babas
  3. Masukkan ayam, kentang dan air kedalam periuk
  4. Siap untuk dihidang
